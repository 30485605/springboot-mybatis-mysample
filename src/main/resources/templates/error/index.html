Error Handling Sample

Handle by Controller themself
コントローラ・クラスそのものに実装された例外ハンドラを使用する例.
Handle by Controller advice
別クラスで実装された例外ハンドラ（AOPにより関連付けられる）を使用する例. 例外ハンドラはSampleExceptionHandlerAdviceで実装されている。
Handle by Resolver
例外リゾルバの実装を使用する例. ハンドラ・メソッド内で発生した例外は、 コントローラ独自の例外ハンドラやAOPで関連付けられた例外ハンドラにキャッチされなかった場合、 ハンドラ例外リゾルバ（HandlerExceptionResolverの実装）に処理を委ねられる。
Handle by Default Controller
Spring Bootデフォルトの例外コントローラを使用する例. 前述の3つはいずれもハンドラ内で発生した例外を処理するもの。 そもそもマッピングに失敗した場合などに発生す例外はこの例外ハンドラ・レベルで処理することになる （ExceptionHanderインターフェースの実装によりカスタマイズが可能）。
